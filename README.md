# hallym_calendar
2024년 하계 계절학기 모바일 프로그래밍 과목 기말 과제 입니다.<br>
첫 플러터 프로젝트입니다.<br>
<br>

## 프로젝트 설명
- 챗봇이 있는 투두리스트 입니다. <br>
<br>

## 프로젝트 기능
1. **구글 로그인 기능**과 **게스트 로그인 기능**이 나뉩니다. <br>
2. **라이트모드/다크모드**를 선택할 수 있습니다.<br>
3. **일정을 등록**할 수 있습니다.<br>
4. **기록된 일정을 확인하고 체크**할 수 있습니다.<br>
5. **캘린더**를 통해 일정을 체계적으로 관리할 수 있습니다.<br>
